What will you be learning?

  • Basic techniques jQuery - implementation script
  • Selectors - select elements on the page
  • Events - track down and handle events (e.g. a mouse click, hover over an image,…)
  • Effects - fade, slide, toggle, and so much more fun stuff
  • jQuery UI - add and configure user interaction elements
  • Animation with jQuery
  • How to select, add and configure plugins
  • JSON and AJAX - reading and writing external data, without doing a page refresh
  • Define a custom plugin

How long will it last?

If you decide to go for presencial learning in the CVO de Verdieping, through the Full Stack WebDeveloper, this course has a duration of around two 10 months.
